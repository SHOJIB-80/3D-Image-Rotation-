# 3D-Image-Rotation-

## Overview
3D Image Rotation is a web-based project designed to showcase 3D rotation of images using CSS and HTML. The project demonstrates how to manipulate images in a three-dimensional space, allowing users to interact and view images from different angles.

## Features
- 3D rotation of images
- Interactive user interface
- Responsive design

## Technologies Used
- CSS: 74.4%
- HTML: 25.6%
